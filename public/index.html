About AI Toolkit

Features

  • Web Search Agent: Search the web and generate research documents
  • Image Generation: Generate images using OpenAI DALL-E
  • Design Feedback: Screenshot websites and get AI design feedback

How to Use

  1. Click on a tool from the Tools section
  2. Follow the prompts in your terminal
  3. Results will be saved to output directories

Output Directories

  • references/ - Web search results
  • references/ai_feedback/ - Design feedback
  • images/ - Generated images
